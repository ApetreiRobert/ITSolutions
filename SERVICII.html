    • Mentenanta Computere
    Servicii complete de mentenanta computere atat software cat si hardware. Curatare de praf, rezolvare probleme hardware, optimizare sistem de operare, update-uri de securitate si backup de date.
    Orice computer care nevoie la un moment dat de servicii de intretinere pentru a preveni pierderea de performanta in timp sau posibile defectiuni ce vor intrerupe activitatea de lucru. Serviciul de intretinere sau mentenanta are ca obiectiv mentinerea computerului in paramerii optimi de functionare si siguranta informatica, pentru a preveni pierderi de date sau defectiuni. Frecventa nevoii de mentenanta depinde de intensitatea lucrului pe acel computer, programe folosite cat si mediul in care functioneaza.
    • Curatare de praf daca este necesar - Curatare completa de praf, inlocuire de pasta termo-conductoare si pad-uri, verificare contacte si inspectie vizuala componente.
    • Verificare ventilatoare - Inlocuirea ventilatoarelor uzate sau defecte si lubrifierea lor daca este posibil cu ulei siliconic.
    • Optimizare software - Reinstalarea sistemului de operare daca este cazul si optimizarea lui pentru performante cat mai bune. Scanare de errori, optimizare registri, defragmentare HDD, optimizare programe.
    • Actualizari de securitate - Verificam status-ul si istoricul antivirusului, facem o scanare completa de virusi si ne asiguram ca antivirusul este actualizat si functional. Verificam si actualizam sistemul de operare cat si anumite programe precum browserele de internet.
    • Rezolvare de probleme - Probleme hardware si software raportate de client.
    • Analiza de performanta - Evaluam performanta in lucru al computerului si facem sugestii de upgrade daca este cazul.
